# anfeespi-portfolio

Personal portfolio of Andrés Espitia (anfeespi), built with React + TypeScript + Vite.

This is the React migration of the original Angular portfolio, keeping the exact same look & feel.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The production bundle is generated in `dist/`.
